# MoringaPure 🌿

A mobile-friendly e-commerce landing page for premium organic Moringa powder.

## Features
- ✅ Fully responsive (mobile + desktop)
- ✅ Sticky navbar with mobile hamburger menu
- ✅ Hero section with product visual
- ✅ Benefits, product pricing, and add-to-cart
- ✅ Customer reviews section
- ✅ Contact form with validation
- ✅ Clean green-themed UI

## Tech Stack
- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript

## Files
| File | Description |
|------|-------------|
| `index.html` | Main website structure |
| `styles.css` | All styling and responsive layout |
| `script.js` | Cart, form, and menu interactions |

## Live Preview
Open `index.html` in any browser to view the website.
